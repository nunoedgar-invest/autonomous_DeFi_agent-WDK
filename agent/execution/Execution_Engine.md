# Refer to ececution_engine.py

Your execution_engine.py should orchestrate the full pipeline:

```Bash
Decision Engine
      │
      ▼
Execution Engine
      │
 ┌────┼───────────────────┐
 ▼    ▼                   ▼
Build TX   Estimate Gas   Sign TX
      │
      ▼
Submit to Blockchain
      │
      ▼
Update Agent State
```

# Example Action Flow

Action generated by your strategy:

```Python
action = {
    "type": "swap",
    "protocol": "uniswap",
    "protocol_address": "0xUniswapRouter",
    "from_token": "USDT",
    "to_token": "ETH",
    "amount": 100,
    "chain_id": 1,
    "calldata": "0xabc123..."
}
```

Execution:

```Python
execution_engine.execute(action, agent_state)
```

Output:

```Python
[ExecutionEngine] Executing action
[ExecutionEngine] Transaction built
[ExecutionEngine] Gas estimated: 210000
[ExecutionEngine] Transaction signed
[ExecutionEngine] TX sent: 0x8391ab...
[ExecutionEngine] Agent state updated
```
